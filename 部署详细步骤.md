# 🚀 部署详细步骤

## 第一部分：推送代码到 GitHub

### 步骤 1：打开 PowerShell

1. 按 `Win + X`
2. 选择 **"Windows PowerShell (管理员)"** 或 **"终端"**

### 步骤 2：进入项目目录

复制粘贴这个命令到 PowerShell：

```powershell
cd "C:\Users\baiyuzhuo\Desktop\B站视频下载"
```

按 Enter

### 步骤 3：查看状态

```powershell
git status
```

按 Enter，你会看到待提交的文件列表。

### 步骤 4：推送到 GitHub

```powershell
git push -u origin main
```

按 Enter

#### 如果要求输入密码/Token：

**方法A：使用 GitHub Token（推荐）**

1. 访问 https://github.com/settings/tokens/new
2. 输入 Token 名称：`vercel-deploy`
3. 勾选 **"repo"** 权限
4. 点击 **"Generate token"**
5. **复制生成的 Token**（只显示一次！）
6. 回到 PowerShell，粘贴 Token 作为密码
7. 按 Enter

**方法B：使用 GitHub 密码**

直接输入你的 GitHub 账户密码（不会显示）

等待推送完成（会显示 `done` 或 `✔️`）。

### 步骤 5：验证

访问 https://github.com/BAYMAX046/bilibili-asr-tool 检查代码是否上传成功。

---

## 第二部分：在 Vercel 上部署

### 步骤 1：登录 Vercel

1. 访问 https://vercel.com
2. 点击右上角 **"Sign In"** 或 **"Login"**
3. 选择 **"Continue with GitHub"**（快速登录）

### 步骤 2：导入项目

1. 登录后，点击左上角 **"New Project"** 或 **"Add New"** → **"Project"**
2. 在 **"Import Git Repository"** 部分
3. 粘贴你的仓库链接：
   ```
   https://github.com/BAYMAX046/bilibili-asr-tool
   ```
4. 点击 **"Import"**

### 步骤 3：配置项目设置

页面会显示一个表单。**什么都不用改**，直接往下滚。

### 步骤 4：添加环境变量（重要！）

向下滚动，找到 **"Environment Variables"** 部分。

点击 **"Add"** 或 **"+"** 按钮，添加以下 4 个变量：

#### 第1个变量
- **Name**: `OSS_REGION`
- **Value**: `oss-cn-shanghai`
- 点击 **"Add"**

#### 第2个变量
- **Name**: `OSS_ACCESS_KEY_ID`
- **Value**: 粘贴你的 AccessKey ID
- 点击 **"Add"**

#### 第3个变量
- **Name**: `OSS_ACCESS_KEY_SECRET`
- **Value**: 粘贴你的 AccessKey Secret
- 点击 **"Add"**

#### 第4个变量
- **Name**: `OSS_BUCKET`
- **Value**: `asr-tool-bucket`
- 点击 **"Add"**

### 步骤 5：部署

1. 所有环境变量都添加完后
2. 点击页面底部的 **"Deploy"** 按钮
3. 等待部署完成（通常 2-3 分钟）

你会看到进度：
```
✓ Build complete
✓ Ready for production
```

### 步骤 6：获取链接

部署完成后，页面会显示你的公网链接，格式是：
```
https://bilibili-asr-tool.vercel.app
```

（或可能是其他名称，取决于你的项目配置）

## 常见问题

### ❓ 部署失败怎么办？

1. 检查环境变量是否全部填入
2. 检查 GitHub 仓库是否确实有代码
3. 点击 Vercel 页面上的 **"Logs"** 查看错误信息
4. 可能需要重新部署：点击 **"Redeploy"**

### ❓ 如何更新代码？

1. 在本地修改文件
2. 执行：
   ```powershell
   cd "C:\Users\baiyuzhuo\Desktop\B站视频下载"
   git add -A
   git commit -m "更新说明"
   git push
   ```
3. Vercel 会**自动重新部署**

### ❓ 怎么让其他电脑访问？

直接分享这个链接：
```
https://bilibili-asr-tool.vercel.app
```

不需要 VPN，全球都能访问！
